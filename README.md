Sorting Algorithm Performance Analyzer (DAA Project)

This is a simple C++ project that compares how different sorting algorithms perform.

It measures:

Execution time (in milliseconds)
Number of comparisons
Number of swaps
Option to save results to a file

This project is made for DAA lab work and learning how sorting algorithms behave on different inputs.

Features

>Modular code structure
Command line based program
User can enter array manually or generate data
Multiple sorting algorithms included
Performance metrics displayed clearly
Easy to understand and modify
Good for viva and demo

>Sorting Algorithms Included

Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
Count Sort

>Metrics Measured

Metric	Meaning
Execution Time	Time taken to sort the array
Comparisons	How many element comparisons were made
Swaps	How many element swaps/movements happened
